# Smart-BMI-and-Health-risk-Analyzer
This project is a Python-based command-line application developed as part of an AI/ML internship task. It collects a patient's basic information, calculates their Body Mass Index (BMI), determines their BMI category, assesses health risk, and generates a health report.

# Features
Collects patient information (Name, Age, Weight, Height)
Calculates BMI using the standard formula
Classifies BMI into health categories
Assesses health risk based on BMI and age
Displays personalized health recommendations
Includes Daily Water Intake Calculator (Challenge 1)
Includes Weight Difference Calculator (Challenge 2)

# Technologies Used
Python 3
Google Colab
Arithmetic, Relational, and Logical Operators
Conditional Statements (if, elif, else)
Formatted Output using f-strings
